The description of 텍뷰(텍스트뷰어)

텍뷰는 텍스트 파일(.txt)로 된 무협지 또는 소설을 보기 위한 용도로 만들었습니다.
광고 없이 간단한 기능만 필요하신 분들에게 적합합니다.

[권한설명]
- 오디오 설정 변경
- 블루투스 기기와 페어링
=> [읽어주기] 모노 블루투스 지원을 위해 필요
- 휴대전화 상태 및 ID 읽기
=> 읽어주기 중 전화올때 읽어주기 실행을 중지하기 위해 필요
(중지하지 않으면 읽어주기 소리와 통화 소리가 같이 들림)
- USB 저장소의 콘텐츠를 수정/삭제 SD 카드의 콘텐츠를 수정/삭제
=> [파일 열기]창에서 파일 삭제할 때 필요

[주요특징]
- 광고가 없습니다.
- 절전기능으로 인한 화면 꺼짐을 방지합니다.
- 파일 인코딩을 자동 판단합니다. (한글, UTF-8, UTF-16)
- 화면에 많은 글을 표시하기 위해 원문을 살짝 조작합니다.(줄붙임 처리)
- 읽어주기(TTS) 기능이 있습니다.(한국어TTS 설치 기기만 동작함)
- 모노 블루투스헤드셋으로 읽어주기를 들을 수 있습니다.
- 창모드 가능(설정 → [창모드]에서 [창모드 사용] 선택)

[설정 및 기능]
- 화면밝기, 폰트(외부폰트가능), 글자크기, 줄간격, 글자색, 배경색(이미지) 설정
- 페이지 이동, 찾기, 북마크, 자동 스크롤 기능
- 전체화면 보기, 타이틀바 숨김, 상태(알림)표시줄 열기 기능
- 마지막 보던 위치 기억
- 비밀번호 잠금 기능

자세한 기능 설명은 앱에서 메뉴 > 도움말을 참조하세요.

[제약사항]
- 용량이 큰 텍스트파일은 지원하지 않습니다.
- 앱 실행 중 메모리 부족 에러로 앱이 종료될 수 있습니다.
(갤럭시S2에서 2MB까지만 확인함. 그 이상은 보장 못합니다.)
- 에러 없이 원활히 이용하시려면 파일을 최대 1MB이하로 줄이고 메모리정리도 가끔 해주세요.
